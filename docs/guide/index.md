# 新人指引

欢迎加入**虚幻工作室**。这份指引帮助你快速入室、配置环境、了解规范，在第一天就能顺畅地参与项目开发。

## 本栏目包括

| 页面 | 内容 |
|------|------|
| [入职流程](./onboarding) | 入室登记、权限开通、仓库配置 |
| [常用工具](./tools) | 推荐开发工具与配置方案 |
| [开发规范](./ue-style) | 虚幻引擎 C++ / 蓝图命名与编码规范 |

## 你需要准备

- 一台运行 **Windows 10/11** 的 PC（虚幻引擎目前对 Linux/macOS 支持有限）
- 一个 **GitHub** 账号
- 至少 **150 GB** 可用磁盘空间（UE5 本身约 60–90 GB，加上项目内容）
- 一颗愿意折腾的心 —— 遇到问题先查文档，再问前辈

## 快速开始

```bash
# 克隆本 Wiki 仓库（文档站本身）
git clone https://github.com/yuyixiaolan/UnrealStudio-Wiki.git

# 安装依赖并本地预览
cd UnrealStudio-Wiki
npm install
npm run docs:dev
```

> 如果这是你第一次接触虚幻引擎，建议先通读[学习资源](/resources/)中的入门教程。

## 提问的智慧

在群里提问之前，建议先：
1. 搜索本 Wiki —— 很可能已经有人踩过同一个坑
2. 粘贴报错信息，而不是只发「报错了」
3. 说明你尝试过哪些方法

好的提问 = 好的回答。
