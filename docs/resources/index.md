# 学习资源

工作室成员积累的推荐学习资料。从入门到进阶，覆盖蓝图、C++、渲染、动画、Gameplay 等多个方向。

## 入门

适合第一次接触虚幻引擎的同学。

| 资源 | 类型 | 推荐理由 |
|------|------|----------|
| [Unreal Engine 5 官方初学者系列](https://dev.epicgames.com/community/learning/courses) | 视频 | 官方出品，最正统 |
| [UE5 蓝图入门（B站）](https://www.bilibili.com/) | 视频 | 中文讲解，门槛低 |
| [Unreal 官方文档](https://docs.unrealengine.com/5.4/) | 文档 | 最权威的参考来源 |
