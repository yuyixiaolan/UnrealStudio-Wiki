## 存档结构

往届资料建议提交：

```
archive/
└─ 2025-1/
   ├─ 项目源码 (Git 仓库)
   ├─ 可执行包
   ├─ 项目文档 (Markdown)
   ├─ 展示 PPT / 视频
   └─ 传承文档.md   ← 给下一届的话
```

## 传承文档模板

如果你在结项后还有话想对下一届说，请写一份 `传承文档.md`。不需要格式，可以是一段话，也可以是列表：

> **给学弟学妹们：**
>
> 1. 我们踩过最大的坑是 xxx，一定记得 xxx。
> 2. 这个插件不要用，有内存泄漏。
> 3. 建议从 xxx 开始做，而不是 xxx。
> 4. 遇到问题先看官方文档，比任何教程都准。

一份好的传承文档比代码更有价值。
